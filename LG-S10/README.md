# Le Genepi (LG) environmental monitoring systems
Column definitions:

1. Station ID
2. Year
3. Month
4. Day
5. Hour
6. Minute
7. Second
8. Time since the epoch [s]
9. Sequence Number
10. Ambient Temperature [°C]
11. Surface Temperature [°C]
12. Solar Radiation [W/m^2]
13. Relative Humidity [%]
14. Soil Moisture [%]
15. Watermark [kPa]
16. Rain Meter [mm]
17. Wind Speed [m/s]
18. Wind Direction [°]

Source: EPFL repository
